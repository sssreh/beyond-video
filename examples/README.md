# Examples

The examples demonstrate how to use the library.

## 01_connect.py

Connect and disconnect.

## 02_list_recordings.py

Retrieve the recording list.

## 03_download_latest.py

Download the latest recording.

## 04_resume_download.py

Demonstrate resumable downloads.

## 05_show_version.py

Retrieve firmware information.

## 06_raw_cgi.py

Execute a CGI request and display the raw response.
